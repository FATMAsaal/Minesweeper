minesweeper module¶

Ce module définit les fonctions permettant de

  • créer un jeu de démineur,

  • accéder à quelques états possibles d’une case,

  • accéder à quelques états possibles du jeu.

Il nécessite le module cell.

Définition d’un type énuméré¶

  • Une classe pour définir un type énuméré décrivant les trois états possibles d’un jeu.

    class minesweeper.GameState[source]¶

    A class to define an enumerated type with three values :

    • winning

    • losing

    • unfinished

    for the three state of minesweeper game.

Constructeur¶

  • La fonction qui crée un jeu de démineur :

    minesweeper.make_game(width=30, height=20, nbombs=99)[source]¶

    return a minesweeper game of size width*height cells with nbombs bombs.

    Parameters

    • width (int) – [optional] horizontal size of game (default = 30)

    • height (int) – [optional] vertical size of game (default = 20)

    • nbombs (int) – [optional] number of bombs (default = 99)

    Returns

    a fresh grid of width*height cells

    UC

    0 < width, height and 0 <= nbombs <= width*height

Divers¶

  • Une fonction pour obtenir les coordonnées des cellule voisines d’une cellule.

    minesweeper.neighborhood(x, y, width, height)[source]¶

    return the list of coordinates of the neighbors de cell (x,y) in a grid of size width*height

    Parameters

    • x (int) – x-coordinate of a cell

    • y (int) – y-coordinate of a cell

    Returns

    Return type

    list of tuple

    UC

    0 <= x < width and 0 <= y < height

    Examples

    >>> neighborhood(3, 3, 10, 10)
[(2, 2), (2, 3), (2, 4), (3, 2), (3, 4), (4, 2), (4, 3), (4, 4)]
>>> neighborhood(0, 3, 10, 10)
[(0, 2), (0, 4), (1, 2), (1, 3), (1, 4)]
>>> neighborhood(0, 0, 10, 10)
[(0, 1), (1, 0), (1, 1)]
>>> neighborhood(9, 9, 10, 10)
[(8, 8), (8, 9), (9, 8)]
>>> neighborhood(3, 9, 10, 10)
[(2, 8), (2, 9), (3, 8), (4, 8), (4, 9)]
