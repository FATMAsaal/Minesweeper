cell module¶

Ce module définit les fonctions permettant de

  • créer une cellule (case) du jeu de démineur,

  • accéder à quelques attributs d’une cellule,

  • modifier quelques attributs d’une cellule.

Constructeur¶

  • La fonction qui crée une cellule :

    cell.make()[source]¶
    Returns

    a new hidden cell of a minesweeper’s grid. existence of a bomb, number of bombs in neigh_borhood have to be stated later.

    Return type

    cell

    UC

    none

    Examples

    >>> cel = make()
>>> is_bomb(cel)
False
>>> is_hypothetic_bomb(cel)
False
>>> is_revealed(cel)
False
>>> number_of_bombs_in_neighborhood(cel)
0
